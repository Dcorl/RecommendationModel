# RecommendationModel
## Movie &amp; TV Shows Recommendation Model:

A content-based recommendation system that suggests similar movies and TV shows using key features such as title, genre, and description.

## Development Environment And Resources
### Programming Language:
- Python

### Programming language Libraries:
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Squarify

### Hardware
- Device: iMac
- Processor: Intel Core i5
- Memory: 8 GB 2400 MHz DDR4

### Operating System
- Operating System (OS): macOS – Ventura 13.7.8

### Software
- Integrated Development Environment (IDE): PyCharm 2024.1.4 (Professional Edition)
- Python Version: Python 3.12

### Database Management System
No database management system is required; instead, a CSV dataset is sufficient. The dataset is from the Kaggle platform and will be imported into the standalone application.

### Data Source
- Platform: www.Kaggle.com 
- Dataset: Netflix Movies and TV Shows
- Link : https://www.kaggle.com/datasets/shivamb/netflix-shows

## User Guide

### Steps to run
1. Download or clone the repo

2. Unzip the project to create a folder for the project from the Zip file.

3. Open the extracted project folder in an IDE of your choice (PyCharm was used during development)

4. If necessary, configure your Python interpreter to Python 3.12 to ensure compatibility with project dependencies.

5. Open the IDE terminal in the project's root directory, and please install the following dependencies with these commands. 

- pip install streamlit
- pip install pandas
- pip install -U scikit-learn
- pip install matplotlib
- pip install seaborn
- pip install squarify


6. To run the project, open the IDE terminal in the project's root directory and run this command
streamlit run main.py


7. After running this command, the application will open in a web browser.
